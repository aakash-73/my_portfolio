# 🎨 AN Portfolio

A bold, modern portfolio featuring geometric design elements, custom sticker branding, and a striking black/white/pale-green/red color scheme. Built for Full Stack Developers who want to stand out.

## 🚀 Features

- **Visual Identity**: Animated geometric logo with initials (AN).
- **Design System**: Sharp-cornered cards, progress bars, and geometric timeline.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Animations**: Smooth Framer Motion transitions and hover effects.

## 💻 Tech Stack

- **Framework**: React 18
- **Animations**: Framer Motion
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Icons**: React Icons

## 📦 Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
